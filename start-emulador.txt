  Run instructions for iOS:
    • cd Nubank && react-native run-ios
    - or -
    • Open Nubank\ios\Nubank.xcodeproj in Xcode or run "xed -b ios"
    • Hit the Run button

  Run instructions for Android:
    • Have an Android emulator running (quickest way to get started), or a device connected.
    • cd Nubank && react-native run-android
